# Competitor Benchmark Seeds

Use this as a starting point for inspiration, then adapt to product context.

## Products and Reusable Patterns

1. Asana
- Pattern: timeline/list/board views with clear context switching.
- Pattern: goals + progress views linked to execution.
- Source: https://asana.com/features

2. ClickUp
- Pattern: multi-view workspace and customizable productivity dashboard.
- Pattern: docs + tasks + chat in one workflow.
- Source: https://clickup.com/features

3. Monday.com
- Pattern: flexible boards, automation recipes, and role-based dashboards.
- Pattern: workflow visibility for cross-functional teams.
- Source: https://monday.com

4. Notion
- Pattern: modular blocks, database-backed pages, and wiki-task unification.
- Pattern: content-first UI with composable surfaces.
- Source: https://www.notion.com/product

5. Linear
- Pattern: speed-focused issue workflow, cycles, triage, and keyboard-first interactions.
- Pattern: opinionated, low-friction navigation.
- Source: https://linear.app/features

## Benchmark Method

For each competitor, score 1-5 across:
- onboarding clarity
- navigation depth vs simplicity
- speed of task completion
- collaboration affordances
- dashboard signal quality
- visual polish and consistency

Summarize what to emulate and what to avoid.
